# Quick List - Todo Application

A simple, elegant, and responsive todo list application built with vanilla JavaScript, HTML, and CSS. Keep track of your tasks with local storage persistence.

![Quick List](./assets/img/logo_img.png)

## Features

✨ **Core Features:**
- Add new tasks quickly
- Mark tasks as complete/incomplete with checkboxes
- Delete tasks
- Persistent storage using localStorage
- Clean and intuitive user interface
- Fully responsive design
- Keyboard support (Enter key to add tasks)

## Demo

Visit the live application: [Quick List](https://deceptiveneeraj.github.io/quicklist/)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/deceptiveneeraj/quick-list.git
```

2. Navigate to the project directory:
```bash
cd quick-list
```

3. Open `index.html` in your browser or use a local server.

## Usage

1. **Add a Task:** Type your task in the input field and either click the "Add" button or press Enter
2. **Complete a Task:** Click the checkbox next to the task to mark it as complete
3. **Delete a Task:** Click the "Delete" button to remove a task from the list
4. **Auto-Save:** All tasks are automatically saved to your browser's local storage
